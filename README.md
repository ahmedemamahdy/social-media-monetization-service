# Social Media Monetization Service — Instagram Promo Landing

A prototype landing page for a **4SALE** social-media monetization / Instagram promo
service. Built with the 4SALE design system (brand colors, typography, and tokens).

## Contents

| Path | Description |
|------|-------------|
| `Instagram Promo Landing.dc.html` | The landing page prototype (self-contained HTML). |
| `support.js` | Generated runtime support script for the prototype. |
| `assets/` | Brand logos (`4sale-blue.png`, `4sale-white.png`). |
| `_ds/` | 4SALE design system — tokens, styles, fonts, and manifest. |

## Viewing

Open `Instagram Promo Landing.dc.html` directly in a browser, or serve the folder
locally:

```bash
python3 -m http.server 8000
# then open http://localhost:8000/Instagram%20Promo%20Landing.dc.html
```

## About the design system

The `_ds/` folder contains the 4SALE Design System — the brand and UI system for
[4SALE](https://q84sale.com), Kuwait's largest classifieds marketplace. See
`_ds/4sale-design-system-*/readme.md` for details.
